To install this bookmarklet Grab this icon and drop it in the bookmarks of your browser

Y/N Replacer

Y/N Replacer

It should have created a new Bookmark called "Y/N Replacer".

On certain mobile phone browsers you won't be able to drag & drop the icon, so the installation will be different and you'll need to create manually the bookmark and copy/paste the code in the url field:

Now create a new bookmark in your browser and paste the code in the URL field. (Every browser is different here, so I can't do a proper explaination step by step. reisling has made an incredibly exhaustive guide on this in their own bookmarklet)

Once installed

Check if it's working now!

  • Click on the bookmark while being on this page
  • it should show you a popup with 'Modify With:', insert your nickname!
  • now see if in the following text the y/n have been modified correctly:

"Hey Jane, I'd like you to meet my girlfriend, y/n! (Y/N), this is Jane, Jane, this is [Y/n]!"

Yay!

Reminder that this works ONLY on ao3 and it works only in stories (not in tags or summaries) and it works only with y/n, [y/n] and (y/n) (not with yn or other combinations of characters.)
